# Skill Controller

一个面向 Codex skills 的可视化控制台，目标是让新手也能更直观地查看、理解和管理 skills。

## 这版的区分点

- 可视化管理 skill，而不是只看代码或 prompt
- 中文优先，把英文 skill 说明整理成更容易理解的中文描述
- 支持查看 skill 之间的差异，方便知道“这个 skill 和那个 skill 有什么不同”
- 支持启用、禁用、删除、重命名和个性化设置
- 可查看使用次数、示例效果，以及“不支持修改”的原因
- 选中 skill 后，右侧详情会跟着定位，不用手动找内容
- 提供 `sc` 快捷入口，方便直接打开控制台

## 适合谁

- 刚开始接触 Codex skills 的用户
- 想把一堆 skill 变成可视化面板的人
- 想在 GitHub 上分享、讨论、改进 skill 工作流的人

## 当前状态

- 这是一个可本地预览的 demo，也可以作为后续插件化分享的基础
- 目标是让它更像一个“新手可用的 skill 管理器”，而不是单纯的静态页面

## 欢迎

欢迎使用、修改，也欢迎直接提 issue 或 PR 指点。

## 入口

- `sc`：打开 Skill Controller
